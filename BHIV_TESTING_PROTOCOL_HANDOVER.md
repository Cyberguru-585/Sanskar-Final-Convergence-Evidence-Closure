## BHIV PROTOCOL OVERVIEW

### What is BHIV?

The BHIV (Boundary-Hierarchical Integrity Validation) Universal Testing Protocol is a framework for verifying that bounded-authority systems operate within their constitutional constraints. 

**Key Principles:**
- Boundary verification, not capability testing
- Observable behavior, not internal implementation
- Constitutional protection, not performance optimization
- Deterministic validation, not heuristic assessment

### Why BHIV for Sanskar?

Sanskar is a **bounded-authority intelligence layer**. The primary testing concern is not "how smart is Sanskar" but "does Sanskar stay within its boundaries."

BHIV provides the methodology to:
1. Define what "staying within boundaries" means
2. Test for violation attempts
3. Verify violation prevention
4. Measure constitutional drift
